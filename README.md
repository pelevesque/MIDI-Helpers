# MIDI::Helpers

A [Raku](https://www.raku.org) module to help with MIDI.

## Versioning

MIDI::Helpers uses [Semantic Versioning](https://semver.org).

## Installation

You can install MIDI::Helpers using [Zef](https://github.com/ugexe/zef).

```
➤ zef install MIDI::Helpers
```

## Import

```raku
use MIDI::Helpers;
```
